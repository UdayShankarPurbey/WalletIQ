# WalletIQ

A personal-finance tracker — log income & expenses across multiple wallets, see month-over-month / year-over-year growth, and get nudged when you're overspending.

## Stack
Angular 21 · Node.js + Express 5 · MongoDB · Redis · Cloudinary · Firebase Cloud Messaging · Zod · JWT
